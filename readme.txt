My Bookmarks is a tool for collecting links to websites
with room for comments and a numerically based rating system.

User Stories:

1. Upon loading the app, you can see a list of your bookmarks.
2. When you click on a bookmark, you will see an expanded view,
   showing your description.
3. You can add new bookmarks with title, url, description, and
   rating fields.
4. In the expanded view, you can delete a bookmark.
5. You receive appropriate error messages when a HTML request
   goes wrong.
6. You may dismiss the error message without reloading the page.
7. You can filter the list of bookmarks by minimum star rating.
